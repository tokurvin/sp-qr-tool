How the check letter is calculated

Each character is mapped to a value: 0–9 → 0–9, A–Z → 10–35, - → 36.

A rolling computation runs left to right:
acc = (acc × 31 + char_value) mod 29

The result maps to the output alphabet:
0–25 → A–Z   26 → $   27 → %   28 → *

Multiplier 31 is coprime to 29 (a prime), giving guaranteed detection of all single-character errors and adjacent transpositions — including a hyphen shifting position.
