Code Format Reference
v3 Format
0155SMAPLP1829POFI-00090005-01-15-01J
Batch qty — 4 digits, zero-padded
SKU — 9–11 chars, starts with letter
PO number — PO + 2 letters + hyphen + 8 digits
Shipment # · Total boxes · Box # — all 2-digit
Check letter — A–Z or $ % *
Algorithm acc = (acc × 31 + val(c)) mod 29
Run left to right over all characters (excluding check letter). Map result to output alphabet.
Output alphabet (29 chars · prime) A–Z + $ % *
Prime modulus guarantees 100% detection of single-char errors and adjacent transpositions.
Character values 0–9 → 0–9   A–Z → 10–35   - → 36
Hyphen is included in the computation, so a shifted hyphen is always detected.
Validation regex ^\d{4}[A-Z][A-Z0-9]{8,10}PO[A-Z]{2}-\d{8}-\d{2}-\d{2}-\d{2}[A-Z$%*]$
